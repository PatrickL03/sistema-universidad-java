# Sistema de Gestión Universitaria

Proyecto académico desarrollado en Java utilizando Programación Orientada a Objetos para la gestión de estudiantes, docentes y cursos.

## Funciones
- Registro de estudiantes y docentes
- Gestión de cursos
- Organización modular por clases y paquetes
- Aplicación de herencia y encapsulamiento
- Manejo básico de lógica académica

## Tecnologías
- Java
- NetBeans
- Programación Orientada a Objetos (POO)
